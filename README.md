## ​ Getting Started

```bash
# 1) Clone the repository
git clone https://github.com/miladmo68/Bank-App.git
cd Bank-App

# 2) Install dependencies
npm install

# 3) Run development server
npm run dev

# 4) Build for production
npm run build
